# ⚡ GitHub Codespaces - 30 秒快速啟動

## 🎯 3 步開啟你的公開 URL

### Step 1️⃣ 打開 GitHub Repo

訪問你的倉庫：
https://github.com/HD4ni/gundam-model-advisor

### Step 2️⃣ 啟動 Codespaces

1. 點擊綠色 **< > Code** 按鈕
2. 選擇 **Codespaces** 標籤
3. 點擊 **Create codespace on main**

⏳ 等待 1-2 分鐘...

### Step 3️⃣ 運行應用

在終端執行：
```bash
python app.py
```

## 🌐 獲取公開 URL

✨ 自動彈出通知 → 點擊 **Open in Browser**

或手動獲取：
1. 底部 **Ports** 標籤
2. Port 5000 → 複製 URL
3. 分享給朋友！
